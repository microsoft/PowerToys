# PowerToys v0.99 Release Notes

## Highlights

- **Image Resizer**: Migrated from WPF to WinUI 3, unblocking AOT compilation and aligning the UI with Windows 11 design.
- **Window Manager**: New Grab And Move module enabling Alt+Left Click window dragging and Alt+Right Click window resizing from anywhere on a window.
- **Display Management**: Re-enabled the PowerDisplay module with a new icon, DPI fixes, installer integration, and numerous stability improvements.
- **ZoomIt**: Added panoramic/scrolling screenshot capture, text extraction when snipping, and break timer improvements with screen saver mode.
- **Command Palette**: Added a new pin-to-Dock dialog, Compact Dock mode, persistent calculator history, plain text/image content viewers, and Performance Monitor network unit selection.
- **Quick Accent**: Added subscript/superscript characters, missing Icelandic letter, capitalized superscript N, plus responsive character-set layout in Settings.
- **Advanced Paste**: Fixed auto-copy failing on Electron/Chromium apps like Teams and VS Code by releasing held modifier keys before injecting Ctrl+C.
- **Settings**: Dashboard layout polish, friendly relative dates for last update check, and assorted visual fixes across General, Image Resizer, and Quick Accent pages.
- **Runner**: Added an update-available badged tray icon and a new tray menu entry so pending updates are easier to notice.
- **Keyboard Manager**: Restored reliable multiline text replacement in chat apps and plain editors.
- **Peek**: Auto-detects file name encoding in zip previews, fixing garbled text in archives created on non-UTF-8 systems.
- **General**: Streamlined default module states so new installations start with a lighter initial experience.

## Advanced Paste

 - Eliminated 13 XAML compiler warnings by switching x:Bind expressions on non-observable properties from OneWay to OneTime mode, since these properties never raise change notifications and are refreshed via Bindings.Update() when needed in [#46726](https://github.com/microsoft/PowerToys/pull/46726).
 - Fixed auto-copy failing on Electron/Chromium apps (e.g. Teams, VS Code) by releasing held modifier keys before injecting Ctrl+C in [#46486](https://github.com/microsoft/PowerToys/pull/46486).
