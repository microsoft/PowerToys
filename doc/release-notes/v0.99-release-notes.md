# PowerToys v0.99 Release Notes

## Highlights

- **Window Manager**: Added the GrabAndMove module enabling Alt+Left Click window dragging and Alt+Right Click window resizing, providing Linux-style window manipulation without targeting title bars
- **Display Management**: Re-enabled the PowerDisplay module with a new icon, DPI fixes, UI/UX improvements, installer integration, and follow-up fixes for thread safety, startup restore, and flyout positioning
- **ZoomIt**: Added panoramic/scrolling screenshot capture, text extraction when snipping, and break timer improvements with screen saver mode and optional computer lock
- **Image Resizer**: Migrated from WPF to WinUI 3, unblocking future AOT compilation (estimated 1.5-3x performance improvement) and aligning the UI with Windows 11 design language
- **Command Palette**: Made the Dock window stay on top of all other windows by default, introduced a Compact Dock mode, and added a new pin-to-Dock dialog for richer configuration
- **Command Palette**: Added persistent calculator history with save, reuse, delete, and clear actions, plus configurable primary action (Copy or Paste)
- **Command Palette**: Improved full-screen detection for the activation shortcut, adding rapid-press breakthrough to bypass suppression and live diagnostics in Settings
- **Command Palette**: Added plain text viewer and image viewer content types for extensions, enabling display of raw text and zoomable images directly in the content pane
- **Advanced Paste**: Fixed auto-copy failing on Electron/Chromium apps (e.g. Teams, VS Code) by releasing held modifier keys before injecting Ctrl+C
- **Peek**: Added auto-detection of file name encoding when previewing zip files, fixing garbled text for archives created on non-UTF-8 systems
- **Quick Accent**: Added subscript and superscript Unicode characters to the Special Characters set for keys 0-9, A, E, N, X, Y, Z, and math operators
- **Settings**: Fixed arrow key icons rendering as box placeholders by switching to standard Unicode arrow characters; formatted last update check timestamp as friendly relative dates
