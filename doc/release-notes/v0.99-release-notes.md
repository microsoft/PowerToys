# PowerToys v0.99 Release Notes

## Highlights

- **Light Switch**: New module that toggles Windows light/dark themes with integrated PowerDisplay monitor profile switching.
- **Window Manager**: New Grab And Move module lets you drag windows with Alt+Left Click and resize them with Alt+Right Click (with optional Win modifier), bringing Linux-style window manipulation to Windows.
- **Display Management**: PowerDisplay ships re-enabled with a new icon, DPI fixes, a shared flyout positioning helper, and major cleanup (thread safety, resource leaks, proper volume/brightness init).
- **Image Resizer**: Migrated from WPF to WinUI 3 for a modern Windows 11 look and AOT-ready performance, with JPEG quality and PNG encoder regressions fixed.
- **Command Palette**: Adds persistent calculator history, a pin-to-Dock dialog with Compact Dock mode, Windows Terminal profile pinning, and many reliability fixes for crashes, icons, and focus handling.
- **ZoomIt**: Adds panoramic/scrolling screenshots, text extraction while snipping, and break timer improvements with screen saver mode.
- **Quick Accent**: Adds subscripts/superscripts, a missing Icelandic letter, Shift+N for superscript capital N, and Hungarian additions, plus a fix for the "All available" default parsing.
- **Settings**: Arrow glyphs render correctly in shortcut controls, the Quick Accent character-sets grid now reflows responsively, Dashboard layout is polished, and last-update-check uses friendly relative dates.
- **Runner**: New update-available tray icon badge and tray menu entry so pending updates are easier to notice.
- **Advanced Paste**: Auto-copy now works reliably on Electron/Chromium apps like Teams and VS Code.
- **Peek**: Zip previews auto-detect file name encodings, fixing garbled text from non-UTF-8 archives.
- **Build**: New installations now start with a streamlined default module set, and native Runner defaults match the managed enabled-modules list to avoid first-launch flicker.

## Advanced Paste
