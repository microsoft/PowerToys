# Keystroke Overlay utility

Keystroke Overlay utility provides a way to display keyboard input in Windows PowerToys. This tool helps users who want to display their keyboard input live, such as educators, presenters, and/or visually impaired users.

In order to use the Keystroke Overlay utility, open PowerToys Settings, select the Keystroke Overlay page, and turn on the Enable toggle. 


## How to Activate

Activate by pressing or holding any key you want to display. An overlay will appear displaying that key. If a key is held down, it is treated as several key presses in rapid succession.


## Settings


From the Settings menu, the following options can be configured:


| **Setting**                   | **Description**                                                    |
|-------------------------------|--------------------------------------------------------------------|
| Activation Shortcut           | The shortcut to activate and deactivate the overlay.               |
| Switch Monitor Shortcut       | Quickly move the overlay between monitors.                         |
| Draggable Overlay             | Drag the overlay to the desired position with this setting enabled.|
| Display Mode                  | Switch between various display modes.                              |
| Overlay Timeout (ms)          | Time before the overlay disappears.                                |
| Text Size                     | The size of the text displayed on keystroke.                       |
| Text Color and Opacity        | The color and opacity of the text displayed on keystroke.          |
| Background Color and Opacity  | The color and opacity of the background behind the displayed text. |
