# Hotkey Launcher Module

This module allows users to define custom global hotkeys to launch applications or scripts.

## Features

- User-defined hotkeys
- Launch apps or scripts
- Settings UI for configuration

## Getting Started

1. Enable the module in PowerToys.
2. Open the settings UI to add/edit hotkey-action pairs.
3. Press the configured hotkey to launch the associated app or script.

## Implementation Notes

- Integrates with PowerToys Runner for hotkey registration.
- Uses standard module interface (see core/architecture.md).
